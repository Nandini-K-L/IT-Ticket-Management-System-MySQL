# IT Ticket Management System using SQL

## Overview

This project is designed to manage IT support tickets using SQL and relational database concepts.

## Tools & Technologies

* SQL
* MySQL

## Features

* Ticket Creation
* Status Tracking
* Report Generation
* Query Optimization

## SQL Concepts Used

* Joins
* Aggregations
* Subqueries
* CRUD Operations

## Outcome

Improved ticket tracking and reporting using SQL queries and relational database design.
